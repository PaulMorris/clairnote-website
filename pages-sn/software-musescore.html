An effort is underway to add Clairnote SN support to MuseScore (along with support for other alternative notation systems). Many thanks to Craig Fisher who took this on and has done some great work, putting in a lot of time and effort. The project's code lives at his GitHub repo and he has also blogged about it. Ole Kirkeby has also contributed. The new features have not (as yet) been added to the official MuseScore app, but it is possible to try things out by installing the modified version of MuseScore that is still in development. [caption id="attachment_3065" align="alignnone" width="300"] Clairnote SN in MuseScore (Click to Enlarge)[/caption] While it's still a work in progress, the most basic elements of Clairnote SN are already implemented (the chromatic staff, ledger lines, the 6-6 notehead pattern, etc.). You can create Clairnote SN scores from scratch or convert existing traditional scores to Clairnote SN and back. Some items still to be implemented are double stems to indicate half notes, correct stem lengths, and Clairnote SN's accidental signs and key signatures. If you want to try out the modified version of MuseScore, the system requirements are the same as for MuseScore 2.x.x. If you are using Windows or GNU/Linux you will need to compile the source code that is available from the GitHub repo. (Be forewarned that compiling your own binary app requires a non-trivial level of technical know-how.) If you are using a Mac then you can download a pre-compiled binary app that is simple to install: MuseScore-Clairnote-Mac.zip You will also need the following "staff type template" file (an XML file format) that contains the Clairnote SN settings to load into MuseScore: Clairnote.stt [caption id="attachment_3090" align="alignnone" width="300"] StaffType Templates (Click to Enlarge)[/caption] Please contact us if you have questions about any of this. While LilyPond produces excellent results and is much more feature-complete in terms of supporting Clairnote SN, MuseScore offers the kind of user-friendly graphical user interface that most people now expect. Subscribe to the Clairnote SN blog to stay informed about progress on support for Clairnote SN in MuseScore.
